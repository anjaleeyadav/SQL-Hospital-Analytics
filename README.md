# SQL-Hospital-Analytics

1. Project / Headline
📊 Hospital Database Analytics: Healthcare Operations Intelligence Dashboard

A SQL-based analytics project designed to extract key insights from hospital operations data across multiple locations, departments, and patient records — focusing on patient statistics, departmental performance, and medical expense tracking.

2. Short Description / Purpose
A comprehensive SQL learning project completed during the 30-Days SQL Micro Course by Satish Dhawale (SkillCourse Founder). This project analyzes hospital operational data across multiple hospitals 🏥, 5 departments 🏢, 8 cities 📍, and 1000+ patient records 👥.

The project tracks patient metrics 👨‍⚕️, doctor availability 💼, departmental performance 📊, medical expenses 💰, and operational efficiency ⏱️.

Designed for hospital administrators 📋, data analysts 📈, and healthcare professionals 🩺 seeking data-driven insights.

3. Tech Stack
Database: MySQL, SQL
Functions: SUM, AVG, MAX, GROUP BY, ORDER BY, Date Arithmetic
Dataset: 1000+ Hospital Records, 50+ Hospitals, 8 Cities
4. Features / Highlights
📊 10 Advanced SQL Queries for comprehensive data extraction
👥 Patient Metrics — Total count, distribution by department & city
💼 Doctor Availability — Average doctors per hospital
📈 Top Departments — Identify top 3 departments by patient volume (45% load)
💰 Medical Expenses — Track total (₹1.2M) and monthly trends
⏱️ Length of Stay — Calculate average stay duration (8 days) per department
🗺️ City-wise Analysis — Patient distribution across regions
🔍 Query Optimization — Efficient data retrieval with advanced SQL techniques

6. Business Impact & Insights
📈 Department Optimization — Top 3 departments handle 45% of patient load → improve resource allocation
💰 Cost Management — Total expenses ₹1.2M tracked monthly → identify savings opportunities
📍 Geographic Strategy — City-wise distribution → plan expansion in high-demand areas
⏱️ Efficiency Metrics — 8-day average stay → benchmark against industry standards
📊 Unified Analytics — Single dashboard replaces multiple Excel reports → faster decision-making.
